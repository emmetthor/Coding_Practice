# Path On Grid

## 介紹

若題目與二維地圖有關，而且**移動方向有限制**，就常常可以用這種 DP 解題。

> 若移動方向是四向或八向，就需要使用最短路演算法。

## DP 狀態與轉移

通常以 `dp[i][j] := 抵達 (i, j) 的答案` 當作狀態，~~十分直觀對吧~~。
但是轉移卻有兩種模式：

1. `dp[i][j]` 與**往回一步可以去的地方**有關。
2. `dp[i][j]` 與**所有走 k 步可以去的地方**有關。

這要根據題目去做對應變化。
