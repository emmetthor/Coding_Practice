## 核心想法

將題目條件轉換成點、邊、權重、並用各種圖論演算法解題。

## 常見模型

### 邊依時間加入、消去

可把**時間 $t$ 看成是邊的權重**，並用最短路、生成樹、LCA 解題。

### 使用 xxx 才可以通過特定邊

每個點要分裂成：使用 xxx 0 次、1 次、2 次...。這樣才能避免直接 DFS / BFS 的錯誤。
