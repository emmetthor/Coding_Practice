## 常見模型

### 期望值的線性性

也就是，$E(aX + bY) = aE(x) + bE(y)$，無論常數 $a, b$，也無論 $X, Y$ 是否獨立。

利用這個性質，許多期望值題目可以拆成簡單的子問題再加起來。
