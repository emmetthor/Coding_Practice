# Range dp 區間 dp

## 這個 skill 解決什麼問題？

需要用動態規劃概念維護區間內資訊的 dp。

## 使用時機

子序列問題，凸多邊形三角化問題。

## 核心想法

用 `dp[l][r]` 紀錄區間 $[l, r]$ 的答案，而根據轉移的不同可以分為兩種：

| DP 模式 | 解釋 | 時間複雜度 |
| - | - | - |
| 2D0D | 不需要額外枚舉的區間 DP | $O(n^2)$ |
| 2D1D | 需要 `for (i = l ~ r)` 轉移的 DP | $O(n^3)$ |
