# 倍增

## 這個 skill 解決什麼問題？

函數圖上的快速轉移。

## 使用時機

函數圖，轉移次數 $\ge 10^7$。

## 介紹

倍增的核心觀念是：

$2^k= 2^{k - 1} + 2^{k - 1}$。

透過拆解成二進位，可以將原本需要暴力的 O(n) 壓成 $O(\log n)$。

通常的倍增寫成 `up[i][k]`，也就是在位置 $i$ 做 $2^k$ 次轉移，會到哪裡。

## 常見模型

### 函數圖轉移

由於函數圖上**每個點都只有一條出邊，**因此可以用倍增在上面直接轉移。

::: info 函數圖

函數圖，指的是**所有點都只有一條出邊的特殊圖**。

由於這個性質，因此函數圖的每個連通塊都會是：一個環 + 一些指向環的鍊。

:::
