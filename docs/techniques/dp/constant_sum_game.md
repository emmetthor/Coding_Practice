## 核心想法

定和賽局可以透過兩方分數差轉成「零和賽局」，就可以使用 Minimax 與 Negamax。

## 介紹

分數總和必為定值的遊戲就是定和賽局，而透過轉換可以變成零和賽局。

::: info 轉換

假設兩方的得分為 $A, B$，總和為常數 $S = A + B$。

則兩方的分數差，$A$ 方為 $A - B$，$B$ 方為 $B - A$。

而 $(A - B) + (B - A) = 0$，故定和賽局等價於零和賽局。

:::

### Negamax

Negamax 的核心概念是：**我方的利益就是「負的對方利益」**。在零和賽局中，由於零和的性質，因此可以簡單透過負號轉換來簡化 Minimax 的操作。

因此，Negamax 的通式為：

$$
\text{My Score} = \max_{\text{moves}}\{\text{Gain} - \text{Opponent's Score}\}
$$
