# 難找出與證明的貪心

## 最大相鄰差

::: info 例題

給定一序列 $A = [a_1, a_2, \ldots, a_N], a_i \in [1, \infty)$，請將他排序，使相鄰兩數差的絕對值最大。

:::

### 最大相鄰差貪心方法

將數字按照波峰波谷排序，並用公式與奇偶性質運算。

### 證明

#### N = 3 的解

$$
\max \sum_{i=1}^{n-1} | a_i - a_{i + 1} |
$$

可以先看 $x, A, y$ 一小組。

如果 $A \ge x \wedge A \ge y$，則這部分的答案會是
$$A - x + A - y = 2A - x - y$$

如果 $A \le x \wedge a \le y$，則這部分的答案會是
$$x - A + y - A = -(2A - x - y)$$

如果 $x \le A \le y$，則這部分的答案會是
$$A - x + y - A = y - x$$


如果 $x \ge A \ge y$，則這部分的答案會是
$$x - A + A - y = -(y - x)$$

會發現，若 $A$ 不大不小，那 $A$ 甚至不會對答案造成貢獻，因此最好讓 $A$ 是部分最大值或最小值。

因此，最好的排法會是「... - 波峰 - 波谷 - 波峰 - ...」這種形式。

#### 引理：係數與大小數的關係

::: info 引理例題

若有兩數 $x < y$ 與兩係數 $c_1 < c_2$，那該如何讓數與係數搭配使總和最大？

:::

可用反證法證明 $y$ 配 $c_1$，$x$ 配 $c_2$ 是不好的。

::: info 證明

若使 $c_1y + c_2x$，則與 $c_1x + c_2y$ 相比：

$$
(c_1y + c_2x) - (c_1x + c_2y) = (y - x)(c_1 - c_2)
$$

多出的差必定是正的，因此這樣配會更不好。

:::

因此，可以證明出，**越大的數配越大的係數，越小的數配越小的係數**。

#### N 更大的解

如果 N 是偶數，那把 $A$ 以波浪排法排好後，答案應是：

$$(a_2 - a_1) + (a_2 - a_3) + (a_4 - a_3) + (a_4 - a_5) + \cdots$$

或是反過來，加總後可得：

$$
2(a_2 + a_4 + a_6 + \cdots) - 2(a_1 + a_3 + a_5 + a_7 + \cdots) + a_1 - a_N
$$

透過引理可知：

$a_2$ 這組由於係數 $+2$，應要配 $A$ 中的大數。

$a_1$ 這組由於係數 $+-$，應要配 $A$ 中的小數。

$+ a_1 - a_N$ 這組由於係數 $\pm 1$，應要配 $A$ 中的中間數。

因此，令 $S$ 是排序好的 $A$，N 是偶數的公式如下：

$$
2\sum_{i = 1}^{\frac{N}{2}-1} S_i
+
2\sum_{i = \frac{N}{2}}^{N} S_i
-
S_{\frac{N}{2}}
+
S_{\frac{N}{2} - 1}
$$

如果 N 是奇數，則只差在要把中位數當成是波峰或是波谷。

證畢。
