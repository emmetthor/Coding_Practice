# Micro Scan Line 微掃描線

## 這個 skill 解決什麼問題？

和連續段有關的一維陣列題目。

## 使用時機

連續段。

## 核心想法

用掃描線由左到右掃過一維陣列並記錄資訊。

## 思考流程

先試試使用掃描線並記錄特定資訊可否解題，畢竟微掃描線的解法通常都很好寫。

## 常見模型

### 連續段長度

若要記錄特定連續段長度，可以使用 `counter`，並在遇到預期的連續段項目 `+1`，反之則重置。
