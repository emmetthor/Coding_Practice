## 介紹

把左右分治的過程畫成樹並標上編號，就是線段樹。

這個編號的編法有很多種，而最常見的就是：把樹根當成 1，左邊編號是 $2x$，右邊編號是 $2x+1$。

1. `set(p, x)` 中的 `p` 是 0-based index。
2. `get(l, r)` 中的 `l, r` 是 0-based index，更是 $[l, r)$ 左閉右開區間。

### 節點合併

每次線段樹設定（`set`）與查詢（`get`）時，皆會使用 `op`，也就是節點合併運算。

只要所求的資訊可以透過節點合併求出，那幾乎都能使用線段樹解題。

::: warning 節點合併與交換率

有些資訊不具交換率，也就是 $op(x, y) \not=op(y, x)$，這時**單位元素的設計**與線段樹合併的順序就非常重要。

:::

### 單位元素

`e()` 是單位元素，必須滿足：

$$
\begin{aligned}
op(e(), x) = x\\
op(x, e()) = x
\end{aligned}
$$

在「單點修改，區間詢問」中，也可以想像成**空區間**。
