# Palindrome 回文

## 這個 skill 解決什麼問題？

與回文有關問題。

## 使用時機

回文，也就是 $\forall (l, r) \wedge 1 \le l \le r \le n, s_l = s_r$ 的字串。

## 核心想法

每次看回文字串的**兩端**，再根據字母相同或不同做操作。

## 常見模型

### 預處理回文個數 / 字母變更成回文數

可以使用**漣漪枚舉法**，從字串中**每一個字母**與**每兩個相鄰字母**開始，向外擴張看**兩端**，再做新增或計數。

## 常見錯誤

- 忘記 `...xx...` 是回文，沒有處理中間兩個字母相同的情況。
